# Homelab

Self-configured Ubuntu Server homelab used for learning self-hosting, Docker, networking, and Linux server administration.

---

# Project status

- Nextcloud deployed and working
- CasaOS installed and configured
- AdGuard Home configured for DNS filtering
- Ongoing troubleshooting and networking documentation

---

# Repo structure

- `nextcloud-homelab/` — Nextcloud deployment and configuration
- `casaos-setup/` — CasaOS installation and Docker integration
- `adguard-home/` — DNS filtering and AdGuard Home configuration
- `server-troubleshooting/` — troubleshooting notes and fixes
- `networking/` — networking concepts and configuration notes (planned)
- `security/` — HTTPS, firewall, and hardening notes (planned)
- `screenshots/` — screenshots and service images
